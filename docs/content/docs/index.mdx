# Welcome to Hot Updater

Hot Updater is a self-hostable OTA (Over-The-Air) update solution for React Native apps, serving as an alternative to CodePush.

## Quick Start

This documentation is being migrated to Fumadocs. Content will be populated in the next phase.

## Features

- Self-hosted solution
- Multiple storage providers (AWS S3, Cloudflare R2, Supabase, Firebase)
- Multiple database options (PostgreSQL, Cloudflare D1, Supabase)
- Support for Metro, Re.Pack, and Expo
- Web-based management console
