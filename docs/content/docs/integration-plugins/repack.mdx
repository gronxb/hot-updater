## Introduction
Re.Pack is a modern bundler for React Native.

Hot Updater no longer requires an additional Re.Pack plugin to deploy bundles.

Bundle IDs are generated during the build and deploy flow, and the deployed
archive now includes a `manifest.json` that native code reads at runtime.

## Installation

```package-install
npm install @hot-updater/repack --save-dev
```

## Usage

You can keep your existing `@hot-updater/repack` setup for backward
compatibility, but it is no longer required for new projects.
